# UK Rail Environmental Impact Visualisation (IJC445)

This repository contains the R code and composite visualisation developed for the IJC437 Data Visualisation coursework.

## Project Overview
The project investigates how the environmental impact of the UK passenger rail system varies across operators and over time using ORR data.

## Files in this repository
- `rail_emissions_visualisation.R` – R script used for data cleaning and visualisation
- `figure1_composite_visualisation.png` – Composite figure used in the report

## Data Source
Office of Rail and Road (ORR):  
https://dataportal.orr.gov.uk/data-table-catalogue/

## Notes
Raw datasets are not included and should be downloaded directly from the ORR data portal.
